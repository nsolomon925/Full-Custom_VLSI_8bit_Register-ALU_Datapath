# Full-Custom_VLSI_8bit_Register-ALU_Datapath
A full-custom VLSI implementation of an 8-bit integrated datapath, featuring a transistor-level CMOS cell library, an 8-function ALU, and a 7+1 Register File. This project demonstrates the complete hardware loop: fetching operands from memory, executing complex arithmetic/logic, and performing a write-back to register storage.
